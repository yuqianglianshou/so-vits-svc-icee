# 训练阶段验收清单

这份清单用于当前仓库的单说话人主线训练。

建议在固定测试素材上重复试听，不要每次换样本。这样最容易判断模型到底是在变好，还是只是样本差异造成的错觉。

推荐固定准备 3 组测试素材：

1. 中音区、节奏平稳、咬字清楚的一段
2. 高音较多、转音较多的一段
3. 气声、弱音、句尾处理较多的一段

每次验收时，尽量使用同一组：

- 转调参数
- 推理参数
- 模型后处理参数
- 是否启用扩散 / 检索 / 增强器

## 1 万步左右

### 目标

判断模型有没有走偏，能不能继续训。

### 重点听什么

- 是否已经有明显目标音色方向
- 是否还在严重底噪、电音、破音状态
- 咬字是否完全糊掉
- 高音是否一上去就崩
- 句子是否断裂严重

### 这个阶段常见现象

- 目标音色只有一点点轮廓
- 电音感较重
- 辅音不稳
- 高音区失真较明显
- 细节通常还不成熟

### 什么时候可以继续

如果满足下面任意大部分，就值得继续：

- 能听出目标角色的大致方向
- 没有明显训练崩坏
- 没有大量 NaN / loss 异常
- 中音区已经能基本成句

### 什么时候建议暂停排查

- 完全听不出目标音色方向
- 音质极其刺耳且没有改善迹象
- 大量爆音、断音、卡顿
- loss 明显异常或训练日志报错频繁

## 3 万步左右

### 目标

判断模型是否已经进入“可用雏形”。

### 重点听什么

- 目标音色是否明显变稳定
- 咬字是否开始变清楚
- 中音区是否已经比较自然
- 高音区是否还在大面积破
- 呼吸、尾音、连音是否开始变顺

### 这个阶段常见现象

- 中音区通常会明显比 1 万步成熟
- 咬字会开始成型
- 仍可能有电音感
- 高音和弱音区仍是重点风险区

### 什么时候值得继续长训

- 中音区已经“能用”
- 高音区虽然不完美，但比 1 万步明显进步
- 音色稳定性在持续提升

### 什么时候说明数据/配置可能有问题

- 3 万步和 1 万步听起来几乎没差
- 咬字仍然严重糊
- 串音、跑调、失真依旧非常重

## 5 万步左右

### 目标

判断模型是否已经达到“不错”的区间。

### 重点听什么

- 音色是否已经稳定
- 咬字是否基本清楚
- 高音是否开始站住
- 句尾、转音、弱音是否更自然
- 是否出现了明显的“训练越久越油/越糊”的趋势

### 这个阶段常见现象

- 很多单说话人模型在这里已经进入可用区间
- 中低音通常比较成熟
- 高音、强混响、复杂句尾仍可能是瓶颈

### 什么时候建议继续

- 还能明显听到进步
- 高音区和细节仍在改善
- 试听中经常会出现“已经接近成品”的感觉

### 什么时候可以开始考虑收尾

- 5 万步后提升已经变慢
- 中音区基本满意
- 高频问题只剩小修小补级别

## 8 万步左右

### 目标

判断是否还值得继续投入更多训练时间。

### 重点听什么

- 与 5 万步相比，是否真的还有明显提升
- 高音区是否继续改善
- 咬字是否继续变清楚，还是开始发糊
- 是否出现过拟合倾向
- 是否只是“数值在变”，但听感没明显变好

### 这个阶段常见现象

- 如果数据质量好，这里可能会进一步细化
- 如果数据一般，收益可能已经明显变小
- 有时会出现继续训练但听感不再显著提升

### 什么时候继续

- 明显还能变好
- 你在追求更高上限而不是尽快收工
- 高音、弱音、细节还在稳定改善

### 什么时候建议停

- 和 5 万步相比，提升已经很小
- 推理听感已经满足当前目标
- 继续训练只是增加时间成本

## 每次试听建议固定记录的内容

建议每个 checkpoint 记录下面这些项目：

- checkpoint 名称
- 训练步数
- 使用的测试素材
- 是否启用扩散
- 是否启用检索 / 增强器
- 中音区评分
- 高音区评分
- 咬字清晰度评分
- 电音感/失真程度
- 一句话结论

可以直接写成这种格式：

```text
G_10000.pth
- 中音区：6/10
- 高音区：3/10
- 咬字：4/10
- 电音感：明显
- 结论：有方向，但还早
```

## 什么时候优先怀疑数据，而不是继续加步数

如果你连续几个 checkpoint 都出现这些问题，优先怀疑数据：

- 咬字一直糊
- 高音一直崩
- 音色始终不稳定
- 某些样本始终有明显噪声或混响拖尾
- 同一角色不同片段音色差别太大

这时候继续硬训，通常不如回头检查：

- 切片质量
- 数据一致性
- 噪声与混响
- 音量处理是否统一

## 最后一句建议

不要把“训练多久”当成唯一目标，更重要的是：

- 尽早试听
- 固定样本对比
- 及时止损
- 把时间花在真正还会继续变好的模型上
